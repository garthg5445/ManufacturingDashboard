Manufacturing Operations Dashboard
This is a web application built to help manage a machine shop I named Whistler Precision Manufacturing. It connects a user-friendly interface to a database so shop managers can track machines, live jobs, and inventory in real time. The Database is MySQL and is run on MAMP servers.

1. Machines Page
Shows every machine in the shop, what job it is currently running, and its current status. You can also add new machines here.

2. Jobs Page
Shows a list of all current jobs, which machine they are assigned to, and their status. It includes Edit and Delete buttons to easily update or remove jobs.

🛠️ Built With
Front End: React & Bootstrap (for the website layout and buttons)

Back End: PHP (for handling requests)

Database: MySQL (for saving data)

📦 How to View the Code
All of the code files are packaged together in a ZIP file inside this repository.

Download the repository.

Unzip the manufacturing-dashboard.zip file.

Inside, you will find all the source code files for the front end, back end, and database setup.
